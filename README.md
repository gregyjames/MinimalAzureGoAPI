# MinimalAzureGoAPI

Add the following repository secrets from the Access Keys section of your Azure Container Repo. 
| Name | Description  |
|--|--|
| REGISTRY_LOGIN_SERVER | Registry name in all lowercase. |
| REGISTRY_USERNAME||
| REGISTRY_PASSWORD | |
| REGISTRY_NAME | The name of the registry to push to.|
